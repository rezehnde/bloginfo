== Description ==

A shortcode to get some useful information from your WordPress Installation.

<em>This plugin is dedicated to Esther Miranda Lima, the daughter I didn't have but I miss so much.</em>

== How to use ==

Just type 

<code>[bloginfo $show]</code> 

to get some blog info from your wordpress instalation.

Possible values for $show: <a href="https://developer.wordpress.org/reference/functions/get_bloginfo/#description">here</a>.
